
# 🏦 Bank Login System (C++)

## 📌 Description

This is a basic **Bank Login System** developed in **C++**.
The program allows a user to log in using a predefined **account number** and **password**.

It is a simple console-based project designed to demonstrate basic concepts like:

* Input/Output in C++
* Conditional statements (`if-else`)
* Basic authentication logic

---

## 🔐 Demo Credentials

Use the following credentials to log in:

* **Account Number:** 123456789
* **Password:** 12345

---

## ⚙️ Features

* User login system
* Credential validation
* Error message for incorrect login

---

## 🚀 How to Run

1. Copy the code into a C++ compiler (e.g., VS Code, CodeBlocks, or any online compiler)
2. Compile the program
3. Run the executable
4. Enter the account number and password

---

## 🧠 Concepts Used

* `cin` and `cout`
* `if-else` statements
* Basic program structure in C++
